Midjourney Image Reference Host

How to use this for Midjourney:
  1. Upload your images into the same GitHub repository/folder where this index.html file lives.
  2. Update the filenames in this HTML file to match your uploaded images.
  3. Once pushed to GitHub Pages, right-click any image here, choose "Copy Image Link", and paste it directly into your Midjourney prompt.
